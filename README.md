# a11y checker
> Identifies accessibility issues in HTML markup.


## Install

```bash
npm install --save a11y-checker
```

## Usage

- Import a11yChecker
```js
import a11yChecker from 'a11y-checker';
```

- Call it after page loads:
```js
a11yChecker();
```
